## Adding Compositions Instructions

Renders an MDX page with instructions on how to add compositions

### Component usage

```js
import { AddingCompositons } from '@teambit/react/instructions/adding-compositions';

...
<AddingCompositions />
...
```

### Themed usage

For styling and syntax highlighting this can be wrapped in the MDXLayout Component and Theme Provider.

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { MDXLayout } from '@teambit/ui.mdx-layout';
import { AddingCompositons } from '@teambit/react.instructions.react.adding-compositions';

...
<ThemeContext>
  <MDXLayout>
    <AddingCompositons />
  </MDXLayout>
</ThemeContext>
...
```

### Add a Description and Labels

Add a description for your component and labels which are used for better discoverability of components.

```mdx
---
description: Description of my component
labels: ['label1', 'label2', 'label3']
---
```

### Add some markdown

```mdx
# Add some markdown formated text

List your component features:

- feature 1
- feature 2
```

### Add a Code Block and Live Playground

Add a codeblock to show how your component can be used.

````mdx
import { Hello } from './hello';
```js
() => {
  return <Hello />;
};
``

`
````

Add `live=true` to create a Live Playground:

````mdx
import { Hello } from './hello';

```jsx live=true
() => {
  return <Hello />;
};
```
````
